## Program Details

The MotorPH Payroll System is a streamlined payroll application developed to automate the computation of monthly employee salaries. The system presents employee information and calculates gross and net salary based on precise hours worked and standard Philippine government deductions.

The program reads external employee attendance records, determines the total number of hours worked (factoring in late grace periods and lunch breaks), and strictly applies payroll rules. It automatically calculates the correct brackets for mandated contributions, including SSS, PhilHealth, Pag-IBIG, and Withholding Tax, based on the combined 1st and 2nd cutoff earnings.

This system aims to reduce manual payroll processing, minimize human error, and improve payroll transparency for both payroll staff and employees. It is built entirely using procedural Java, ensuring a lightweight and highly logical execution.
